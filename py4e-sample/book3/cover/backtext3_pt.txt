Python Para Todos
Explorando Dados com Python 3

Esse livro tem como objetivo ensinar pessoas a programar mesmo que elas não
tenha nenhuma experiência anterior. Esse livro pode ser utilizado para ensinar
programação a qualquer pessoa.
